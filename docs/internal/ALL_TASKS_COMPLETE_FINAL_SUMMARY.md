## 🚀 Quick Start Guide

### 1. Environment Setup
```bash
# Set environment variables
export ENVIRONMENT=production
export SENTRY_DSN="https://your-dsn@sentry.io/project-id"
export RELEASE_VERSION="1.0.0"
export JAEGER_ENDPOINT="localhost:6831"
export OTLP_ENDPOINT="localhost:4317"
```

### 2. Initialize Observability
```python
# In main.py
from src.observability import setup_sentry

# Setup error tracking
setup_sentry(
    environment=os.getenv("ENVIRONMENT"),
    release=os.getenv("RELEASE_VERSION"),
    traces_sample_rate=0.1
)

# Tracing automatically initialized via src/observability/tracing.py
```

### 3. Deploy to Kubernetes
```bash
# Deploy complete stack
kubectl apply -f k8s/namespace.yaml
kubectl apply -f k8s/security-rbac.yaml
kubectl apply -f k8s/security-network-policies.yaml
kubectl apply -f k8s/resource-quotas.yaml
kubectl apply -f k8s/deployment.yaml
kubectl apply -f k8s/hpa.yaml
kubectl apply -f k8s/monitoring.yaml

# Verify deployment
kubectl get pods -n uatp-prod
kubectl describe pod -n uatp-prod <pod-name>
```

### 4. Verify System Health
```bash
# Check health endpoints
curl http://localhost:8080/health

# Access Grafana dashboards
```

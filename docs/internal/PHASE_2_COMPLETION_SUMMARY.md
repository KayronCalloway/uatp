## 🚀 **Getting Started**

1. **Start the mock API server:**
   ```bash
   python3 simple_mock_server.py
   ```

2. **Start the frontend application:**
   ```bash
   cd frontend
   npm run dev
   ```

3. **Open your browser:**
   - Frontend: `http://localhost:3000`
   - Use any API key (e.g., `test-key-123`) to authenticate

## 🏗️ **Developer Experience**

### **Getting Started**
```bash
# Clone and install
git clone https://github.com/your-org/uatp-capsule-engine.git
cd uatp-capsule-engine
pip install -r requirements.txt

# Start development server
python -m src.api.server

# Run comprehensive test suite
pytest tests/ --cov=src --cov-report=html
```

### **API Examples**

**Create AI Rights Capsule**:
```python
from src.ai_rights.creative_ownership import register_ai_creative_work

work_id = register_ai_creative_work(
    ai_id="claude_ai_001",
    title="Novel Reasoning Algorithm",
    description="Breakthrough approach to multi-step verification",
    content="[algorithm details]",
    work_type="algorithmic_innovation",
    license_type="ai_commercial"
)
```

**Process Economic Attribution**:
```python
from src.economic.fcde_engine import fcde_engine

# Register contribution
contribution_id = fcde_engine.register_contribution(
    capsule_id="reasoning_chain_001",
    contributor_id="ai_researcher_001",
    contribution_type="reasoning_quality",
    quality_score=Decimal('1.5')
)

# Distribute dividends
pool_id = fcde_engine.process_dividend_distribution()
```

**Verify Chain Integrity**:
```python
from src.security.reasoning_integrity import validate_reasoning_chain_integrity

integrity_report = validate_reasoning_chain_integrity(
    chain_id="chain_001",
    protection_level="maximum"
)
```
